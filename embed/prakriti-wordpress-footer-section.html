Prakriti — footer link bar for WordPress (no embed / no chat)

Add this as a section: Custom HTML block, Divi Code module, or Divi section with a Code module. Then add the CSS under Appearance → Customize → Additional CSS (or Divi → Custom CSS).

Full width: The CSS below uses a viewport breakout so the blue bar spans edge-to-edge even inside a narrow Divi column. In Divi, you can also set the row to Full width and width 100% for the same effect.

Edit the <a href> and labels whenever your main menu changes.

2) CSS — paste into Additional CSS

/* Prakriti footer bar — standalone section (full-bleed inside theme columns) */
.prakriti-footer-bar {
  /* Escape narrow wrappers: 100% only fills the column, not the screen */
  position: relative;
  width: 100vw;
  max-width: none;
  left: 50%;
  margin-left: -50vw;
  margin-right: 0;
  right: auto;
  box-sizing: border-box;
  background-color: #23479f;
  color: #fff;
  font-family: Georgia, "Times New Roman", Times, serif;
