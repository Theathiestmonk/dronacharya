Prakriti AI — WordPress embed (one place for HTML + CSS)

Do not upload this file to WordPress. Copy the two sections below into the right places.

1) Custom HTML block (or Divi Code module) on your embed page

<div class="prakriti-chat-embed">
  <iframe
    src="https://www.prakritischool.ai/?embed=1"
    title="Prakriti AI Assistant"
    loading="lazy"
    allow="clipboard-write; microphone"
  ></iframe>
</div>

2) Where to paste CSS (important)

Best: Appearance → Customize → Additional CSS (or Divi → Theme Options → Custom CSS). The site loads real CSS and there is no broken validator.

Avoid: Divi Code module → Advanced → Custom CSS — its editor often shows false errors like “Expected RBRACE” for calc(), env(), and CSS variables. If the site looks fine, you can ignore the red warnings; if it refuses to save, use Additional CSS instead.
